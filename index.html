Introduction to HTML, CSS, JavaScript & Tailwind

1. HTML (HyperText Markup Language)

HTML is the basic language used to create web pages. With HTML, we can define the content structure such as headings, paragraphs, images, links, tables, and forms. HTML tells the browser what to display on the web page. Each HTML element has a specific tag that indicates its function.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Simple HTML</title>
</head>
<body>
<h1>Hello, World!</h1>
<p>I'm Bella.</p>
</body>
</html>

2. CSS (Cascading Style Sheets)

CSS is used to style web pages. We can set colors, fonts, text size, spacing, layout, shadows, animations, and more. CSS connects to HTML elements through tags, classes, or IDs.

 
  /* Set basic styles for the whole page */
      body {
      background-color: #f0f8ff; 
      font-family: Arial, sans-serif; 
      margin: 0; 
      padding: 20px; 
      }

  /* Style for the main heading (h1) */
      h1 {
      color: #1e90ff; 
      text-align: center; 
      font-size: 36px; 
      }

  /* Style for paragraph text */
      p {
      color: #333;
      line-height: 1.6; 
      text-align: justify; 
      }

3. JavaScript (JS)

JavaScript is a programming language that makes websites interactive. We can add click buttons, animations, forms, and load dynamic data. JavaScript makes websites more lively and responsive.

      <body>
        <h2>JavaScript Example</h2>
        <p id="message"></p>
        <script>
          document.getElementById("message").innerHTML = "Hello World, I'm Bella!";
        </script>
      </body>

4. Tailwind CSS

Tailwind CSS is a modern framework that simplifies styling websites. You can create a neat and responsive design just by adding utility classes.

    <body class="flex items-center justify-center h-screen bg-gray-100">
        <div class="text-center p-4 bg-white rounded shadow">
        <h1 class="text-xl font-bold text-blue-600">Hello!</h1>
        <p class="text-gray-700">Simple Tailwind page</p>
        </div>
    </body>

Conclusion

- HTML ‚Üí Page structure
- CSS ‚Üí Page appearance
- JavaScript ‚Üí Interactivity
- Tailwind CSS ‚Üí Fast and modern styling
Combining these four technologies allows you to create beautiful, functional, and responsive websites.
